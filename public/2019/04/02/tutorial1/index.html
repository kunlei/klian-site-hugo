Apache POI - Excel

 Posted on April 2, 2019  |  Kunlei Lian

I recently worked on a project that required me to interact with Microsoft Excel to obtain input data and present optimization results. It turns out that there is a great package for that purpose and all you need to do is include the following dependencies in the Java project .pom file:

 1<dependency>
 2    <groupId>org.apache.poi</groupId>
 3    <artifactId>poi</artifactId>
 4    <version>3.17</version>
 5</dependency>
 6
 7<dependency>
 8    <groupId>org.apache.poi</groupId>
 9    <artifactId>poi-ooxml</artifactId>
10    <version>3.17</version>
11</dependency>

There are things I do most frequently in my project:

Read information from Excel

For this purpose, there are some simple steps to follow:

  • create a Workbook object using a FileInputStream
  • obtain a reference to the sheet from which you want to read data from
  • create an iterator to loop through the rows
  • retrieve individual cells to obtain needed information

Here is an example code:

 1try {
 2    FileInputStream fileInputStream = new FileInputStream(fullFilename);
 3    Workbook workbook = WorkbookFactory.create(fileInputStream);
 4
 5    Sheet sheet = workbook.getSheet(sheetName);
 6    if (sheet == null) {
 7        System.out.println("There is no sheet named " + sheetName + " in the file!");
 8        return null;
 9    }
10
11    Iterator<Row> rowIter = sheet.rowIterator();
12    while (rowIter.hasNext()) {
13        Row row = rowIter.next();
14        Cell c = row.getCell(0);
15        String value = c.getStringCellValue();
16    }
17
18    fileInputStream.close();
19    workbook.close();
20} catch (Exception e) {
21    e.printStackTrace();
22}

Read and write information to Excel

For this purpose, there is a little more work to do.

  • create a FileInputStream object
  • create a Workbook object binding to the FileInputStream object
  • update the workbook object

  • create a FileOutputStream object and call workbook.write() to write data into file

FileInputStream fileInputStream = new FileInputStream(fullFilename);
Workbook workbook = WorkbookFactory.create(fileInputStream);
fileInputStream.close();

Sheet sheet = workbook.getSheet(sheetName);

I also create a helper function to clear the content in a sheet:

 1public static void clearSheet(Sheet sheet) {
 2    for (int idx = sheet.getLastRowNum(); idx >= sheet.getFirstRowNum(); idx--) {
 3        Row row = sheet.getRow(idx);
 4        if (row != null ) {
 5            sheet.removeRow(row);
 6        } else {
 7            break;
 8        }
 9    }
10}
